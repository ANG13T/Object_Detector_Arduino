# Object Detection Indicators

This projects detects and indicates the proximity of an object using multiple LEDs, a servo, and an ultrasonic sensor

## Materials

- Breadboard
- 8 LEDS
- 8 220ohm resistors
- Arduino Uno
- Servo motor
- Ultrasonic sensror
- Around 20 jumper wires
